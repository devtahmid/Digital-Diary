# Digital-Diary  
![C++](https://img.shields.io/badge/c++-%2300599C.svg?style=for-the-badge&logo=c%2B%2B&logoColor=white)  

The diary provides the following services.

1) Contacts
2) Calender
3) World Clock
4) Games
i) Snake and Ladder
ii) Maze Runner
iii) Tic Tac Toe
5) Horoscope
6) Converter
7) Calculator


Each of these features are implemented as a class in the cpp file
